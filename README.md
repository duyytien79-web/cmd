Hướng dẫn cài đặt và chạy game

Bước 1: Cài Python từ trang chủ python.org

Bước 2: Tải source code từ GitHub : https://github.com/duyytien79-web/cmd.git

Bước 3: Mở project bằng Visual Studio Code

Bước 4: Mở terminal và cài thư viện:
pip install -r requirements.txt

Bước 5: Chạy game:
python main.py
